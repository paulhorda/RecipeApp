# Recipes App

## Features
- Load recipes from the network
- Search recipes
- Sort recipes
- Filter recipes
- Infinite scroll with pagination
- View recipe ingredients
- Save your favorite recipes in a local database
- Share recipes
- Dark theme 
- Localization
- ...
